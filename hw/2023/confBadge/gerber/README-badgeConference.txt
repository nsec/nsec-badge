Information on files for project 
	NORTHSEC 
	BADGE-CONFERENCE 
	2023
===========================================================

Fabrication parameters
-------------------------
* 2 layers FR4 1.6mm
* TOP and BOTTOM copper thickness = 1.0oz 
* Soldermask on both sides = BLUE BACKGROUND
* Silkscreen on both sides = WHITE 
* Lead-free HASL finish
* Tolerance on drills 	=	3mils
* Minimal hole size 	= 	0.25mm
* Minimal isolation	=	5mils
* Minimal trace width	=	10mil
* All vias and slots are plated through holes
* All vias are cover with soldermask


CAD Package
-----------
CadSoft EAGLE 9.6.2 PREMIUM 

Gerber files format
-------------------------------------------------
* RS-274X format
* All files are gerber-X format, except drills (.DRL)

Excellon files format (*.DRL)
------------------------------------
* Excellon format
